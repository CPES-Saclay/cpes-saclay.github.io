# Politique de confidentialité

Le site internet des CPES Paris-Saclay n'utilise aucun cookie interne comme externes pour faire fonctionner le site. Certaines fonctionnalités, comme les vidéos Youtube, demanderont une autorisation individuelle pour fonctionner.
